 🤖 Chatbot AMI

Chatbot desenvolvido com foco no letramento digital de idosos, oferecendo suporte acessível, acolhedor e simples para auxiliar no uso de tecnologias no dia a dia.

---

## 💡 Sobre o projeto

O Chatbot AMI foi criado com o objetivo de reduzir a dificuldade que muitos idosos enfrentam ao utilizar aplicativos, internet e dispositivos móveis.

A proposta é fornecer um atendimento automatizado, claro e humanizado, ajudando o usuário a entender e realizar tarefas tecnológicas de forma mais confiante.

---

## 🚀 Funcionalidades

- 💬 Atendimento via Telegram  
- 🤖 Respostas automatizadas com linguagem simples  
- 🧠 Integração com IA (Google Gemini)  
- 🔊 Saída multimodal: respostas em texto e voz  
- 📲 Suporte para dúvidas do cotidiano digital  
- 💛 Foco em acessibilidade e acolhimento  

---

## 🛠️ Tecnologias utilizadas

- Python  
- Telegram Bot API  
- Google Gemini (IA generativa)  
- Firebase  
- Git e GitHub
